🎯 Основной workflow:

1. 📸 Захват кнопки:
   python3 utils/smart_capture.py
   → Создает templates/button.png

2. 🎯 Создание последовательности:
   python3 sequence_builder.py
   → Создает/редактирует my_sequences.yaml

3. 🚀 Запуск:
   python3 macro_sequence.py --config my_sequences.yaml --run name
